This is my **personal** Sway setup

## Modifications
- Default but French keyboard layout bindings
- Numlock activated by default
- Apps assigned to workspaces
- No windows title
- Custom gaps
- Custom cursor theme
- Wallpaper set to `~/.wallpaper.png`

## Dependencies
- [Futur Dark Cursor](https://www.gnome-look.org/p/1457884)

> Untar to `~/.icons/`

## Installation
Copy config file to `~/.config/sway/config`
```bash
git clone https://github.com/criticalsool/sway-custom.git
mkdir -p ~/.config/sway ~/.icons
cp config ~/.config/sway/config
```

> Install cursor theme in `~/.icons`
